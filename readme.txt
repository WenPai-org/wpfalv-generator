This plugin allows you to add extra scripts to the header and footer of your blog by hooking into wp_head and wp_footer.

== Description ==

= Easily Insert WPTongji Code =

WPTongji Code is a simple plugin that lets you insert code like Baidu Statistics, Google Analytics, custom CSS, Facebook Pixel, and more to your WordPress site header and footer. No need to edit your theme files!

The simple interface of the WPTongji Code plugin gives you one place where you can insert scripts, rather than dealing with dozens of different plugins.

= Features of WPTongji Code =

* Quick to set up
* Simple to insert scripts
* Insert header code and/or footer code
* Add <strong>Baidu Statistics</strong> code to any theme
* Add <strong>Google Analytics</strong> code to any theme
* Add <strong>custom CSS</strong> across themes
* Insert <strong>Facebook pixel code</strong>
* Insert any code or script, including HTML and Javascript

== Installation ==

1. Install WPTongji Code by uploading the `wptongji-code` directory to the `/wp-content/plugins/` directory. (See instructions on <a href="https://WenPai.org/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/" rel="friend">how to install a WordPress plugin</a>.)
2. Activate WPTongji Code through the `Plugins` menu in WordPress.
3. Insert code in your header or footer by going to the `Settings > WPTongji Code` menu.

== Frequently Asked Questions ==

= Can I use WPTongji Code to install Baidu Statistics? =

Yes, you can insert your Baidu Statistics code in the `Scripts in Header` field.

= Can I use WPTongji Code for Baidu Ads? =

Yes, to verify your account or to tag your page for Auto ads, paste the code Ads gives you, into the Scripts in Header field.

= How to disable WPTongji Code on a specific page? =

You can use one of the three available boolean filters: disable_ihaf, disable_wptc_footer, disable_wptc_header, disable_wptc_body and return true value in order to disable printing either on the entire page or specifically in the header or footer of a given page.
